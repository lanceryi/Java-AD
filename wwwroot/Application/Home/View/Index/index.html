不便来公司合作方式

依照《中华人民共和国合同法》保证合作双方在公平、公正、公信、公开的基础上，总部在全国实行统一合作条件及标准。如果因为各种原因不方便来公司考察，可异地直接接产合作。

为了更方便外地合作者合作，总部在发货同时，标准配备：一式两份（加盖红色公章）签约合同书、VCD教学光盘、编号图纸。模具化生产，简单易学，一看就会。另发原料30－1000件，产品一律由总部统一现金收购。首次申请合作需交纳合同保证金2800元（注*此款在产品交货时一律退还，后期不再有任何费用）总部长期免费为您发货，并预付您下批产品加工费的50%以利于加工户扩大经营、安心生产，让您赢在起点。

注：首批发货30件，等产品上交后，从 第二批开始走入正轨，长期发货每批1000件。也可根据加工方的实际情况自行计划取货数量，工人多可以多取货，工人少可以少取货，最少30件即可发货。

例如： 接产“生意兴隆·钻石画”按每月发货量500件计算，加工费共计119000元，总部将提前预付您59500元的加工费（即下批总加工费的50%）当日银行汇款到您名下，余款收货时清。

为什么暂收加工户的保证金？何时退还？

中国地域辽阔，总部在诚征加工户的过程中，事实上不能对每个申请合作的投资者进行商业信誉的独立调查和评估。在公正、公平、公开的合作基础上，为了双方的合法权益不受到侵犯并且保证加工产品必须返还总部，确保双方健康、稳定、互惠、互利、携手发展，总部将采取暂收合同保证金的合作方式，首批产品生产完即无息退还。

不便来公司合作者常见问题？

本企业珍视商誉、至诚至信。

让合作者100%满意，与您一起成就辉煌。

1、问：我不方便去公司，合同如何签订？

答：总部发货同时，会带有一式两份（加盖红色公章）合同书，自己保留一份，另一份随货寄回总部存档备案。和您亲自来到总部办理接产手续完全一样，免去来往路途劳累以及不必要的差旅费用。

2、问：我们相隔千里，总部如何发货？

答：通过快递公司或邮局EMS发货2-3天左右即到（往返运输费用均由我公司承担）。

3、问：如果在运输过程中有损坏怎么办？

答：专用运输包装、易运输，不易损坏。

4、问：初次合作，刚开始做坏了怎么办？

答：手工钻石画全部为纯手工点钻（粘贴），拿到货后只需按照画面编号粘贴对应钻石即可。

5、问：首批产品交货日期，有时间规定吗？

答：合作者首次生产在制作过程中没有经验，首批产品尽量在1-2个月内完成，如有特殊原因需延期，致电总部延期交货。不过后期会有时间限制，由加工户根据自己的加工能力而定。

6、问：生产出什么样的标准为合格产品？

答：按照画面粘贴对应钻石，外观干净、整齐、粘贴牢固即可。纯模具化生产、完整的工艺流程让您明明白白生产，质量关自己把握。

7、问：把货发过来我们不会加工怎么办？

答：总部发货时，随货会带有VCD教学光盘；模具化生产，简单易学,一看就会。您只是在生产过程中快与慢而己；这点您可以绝对放心。

8、问：产品完成后，如何收货？

答：首批产品完成，可直接发货到公司，货到当即付款；运费由公司承担。也可通知公司上门收货。

9、问：首批产品完成后，从第二批发货开始，为何要先预付50%—80%加工费？

答：预付款是外发加工行业合同法规定。先行预付50%加工费的政策是为了打消合作者的顾虑，以利于扩大经营、安心生产。让您赢在起点。

10、问：每次发货来回时间这么长，我们还要等货到再生产吗？

答：合作走入正轨，合作者每次不用把所有的产品都完成之后再上交，可以预留半个月的货量。以便能保证正常生产。
